# Daneo

A Korean language vocabulary learning app written in JavaScript using React Native for iOS and Android.

## Usage

Install expo globally
```bash
npm install --global expo-cli
```

Clone this repository and in the root folder execute
```bash
expo start
```

Use npm or yarn to install missing modules and packages as needed.
